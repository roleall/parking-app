🚗 Sistema de Gerenciamento de Estacionamento
Aplicação web desenvolvida com React, Vite e Tailwind CSS, criada com foco em aprendizado prático de desenvolvimento front-end moderno, organização de código e simulação de um sistema real de gerenciamento de estacionamento.

O projeto foi idealizado e desenvolvido por um QA com foco em evolução técnica para desenvolvimento, priorizando boas práticas, clareza de código e base sólida para testes e regras de negócio.

🎯 Objetivo do Projeto
Simular um sistema de estacionamento capaz de:

Registrar entrada de veículos

Controlar horários e tempo de permanência

Exibir veículos estacionados

Gerar comprovante de entrada

Servir como base para testes manuais e automatizados

▶️ Como Rodar o Projeto Localmente
Pré-requisitos
Node.js (versão LTS recomendada)

Passos
# Clonar o repositório
git clone <URL_DO_REPOSITORIO>

# Acessar a pasta do projeto
cd parking-app

# Instalar dependências
npm install

# Rodar o projeto
npm run dev
A aplicação estará disponível em:

http://localhost:5173
